Chapter 1. Why We Program

A program is a set of step-by-step instructions you give the computer. Python is the language you write those instructions in. The computer follows them in order, from the top of the file to the bottom.

Words to know

  • Program. A list of instructions.
  • Python. The language you write them in.
  • print. The command that shows text on the screen.

Example

print("Hello, World")
print("I am learning to code")

Try it: Change the words inside the quotes and print two lines about yourself.

Chapter 2. Variables and Expressions

A variable is a labeled box that holds a value so you can use it later. It can hold text or numbers, and you can do math with numbers. The input command lets the program ask you a question and store your answer.

Words to know

  • Variable. A named box that holds a value.
  • String. Text, always inside quotes.
  • Integer. A whole number.
  • input. Asks a question and stores what you type.

Example

name = input("What is your name? ")
print("Hello", name)

age = 30
print("Next year you will be", age + 1)

Try it: Make a variable for your favorite color and print a sentence using it.

Chapter 3. Conditional Code

Sometimes you want the program to make a choice. The word if checks whether something is true. If it is true, the indented lines below it run. The word else covers the other case. Indentation, meaning the spaces at the start of a line, is how Python knows which lines belong together.

Words to know

  • if. Runs code only when something is true.
  • else. Runs code when the if was not true.
  • Comparison. Greater than, less than, or equal to.

Example

hours = 45
if hours > 40:
    print("You worked overtime")
else:
    print("Normal week")

Try it: Change the number in hours and watch which message prints.

Chapter 4. Functions

A function is a named block of code you can reuse. You define it once with the word def, then call it whenever you need it. A function can take an input, called a parameter, and can hand back a result with the word return.

Words to know

  • def. Defines a new function.
  • Parameter. An input the function receives.
  • return. Hands a result back.
  • Call. Running the function by its name.

Example

def greet(person):
    print("Hello", person)

greet("Robert")
greet("Kristina")

Try it: Write a function that prints a short encouragement, then call it twice.

Chapter 5. Loops and Iteration

A loop repeats work so you do not have to write it many times. A for loop walks through a set of items one at a time. A while loop keeps going until something changes. A common pattern is to keep a running total, or to remember the biggest value you have seen so far.

Words to know

  • Loop. Code that repeats.
  • for. Repeats once for each item.
  • while. Repeats until a condition changes.
  • range. Makes a series of numbers to loop over.

Example

for number in range(1, 6):
    print("Counting", number)

Try it: Change range(1, 6) to a bigger number and watch it count.

What Makes It Feel Hard

  • A new way of thinking. You break problems into small logical steps. This is different from how we usually think.
  • New words. Variable, function, loop. These are all new at first, and that is normal.
  • Mistakes show up right away. When something is wrong, the computer tells you. Learning to fix errors takes practice.
  • Repetition matters. You write the same kind of code many times before it feels natural.

Why You Have an Advantage

  • You break big challenges into steps. That is exactly what programming is.
  • You have time to practice. Consistency beats intensity every time.
  • You have a real reason to learn. You want to build tools for your website and your work. That keeps you going.
  • You understand persistence. You have faced challenges far harder than a syntax error. You know how to push through.

Remember This

Coding is not harder than learning to drive or learning an instrument. It just feels different at first. When something feels hard, it usually means you are learning. Every expert started exactly where you are. The only difference is that they kept going.
